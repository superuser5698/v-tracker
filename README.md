# v-tracker
a file tracker like git but it auto commits when file is changed and has like 3 commands
commands(so far) read, init, add and write
read: check what files are being tracked
write: add a file to tracker.txt
init: init a folder by creating v-track and tracker.txt inside
add: add a file to tracker.txt
I have only tested on windows 11
version: Beta
note:I have not yet implmented the auto-commit feature or the commit featur so its kind of useless
